UniWorth Shop Clone

A fully responsive e-commerce web application built with React, Redux Toolkit, and Material UI (MUI).
This project recreates the layout and shopping flow of the UniWorth clothing store with features like category browsing, product display, and cart management.

Features

Responsive Header and Navigation
Includes a mobile drawer menu, category listing with submenus, and hover-based modals for quick navigation.

Product Showcase
Uses Swiper.js sliders to display featured and trending products in a clean layout.

Cart System
Add items to the cart using Redux Toolkit. The cart modal allows users to view and manage their selected products.

Authentication Pages
Signin and Signup pages are built with React Hook Form and Yup for form validation and error handling.

Responsive UI
Built with Material UI components and responsive grid layouts for all screen sizes.

Routing
Uses React Router DOM v6 for page navigation.
